SchedConfigTool
===============

GUI for generating scheduling configuration file


==============
 REQUIREMENTS
==============

WORK IN PROGRESS

Qt SDK.


==============
 COMPILATION
==============

Reach the project folder.

* $ cd src
* $ qmake -o Makefile SchedConfigTool.pro
* $ make


=======
 USAGE
=======

After succesful compiling.

* $ ./SchedConfigTool &
